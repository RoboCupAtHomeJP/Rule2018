- [DSPL] 2018年のルールブックに記載されているOfficial Standard Laptopを使用することができる．
- [DSPL] Official microphoneはないため，各チームが独自で準備したBluetoothマイクを利用することができる．使用する場合は，ロボットには設置せず，手持ちで利用する．
